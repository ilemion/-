# 需求规格说明书(SRS)

## 1. 引言

### 1.1 目的
本文档详细描述云南省企业就业失业数据采集系统的功能需求和非功能需求,作为开发、测试和验收的依据。

### 1.2 范围
系统覆盖云南省、市、企业三级数据采集网络,实现企业基础信息备案、月度就业失业数据上报、多维汇总分析及国家级数据对接。

### 1.3 术语定义
| 术语 | 定义 |
|------|------|
| 建档期 | 企业首次建档时的就业人数基准期 |
| 调查期 | 当前需要填报数据的统计周期 |
| 组织机构代码 | 企业的唯一标识码,最多9位字母数字组合 |
| 市级初审 | 市级管理部门对企业上报数据的初步审核 |
| 省级复核 | 省级管理部门对市级审核通过数据的再次审核 |

## 2. 系统概述

### 2.1 系统用户
| 用户类型 | 职责 | 权限 |
|---------|------|------|
| 企业用户 | 数据填报、信息查询 | 本企业数据填报、历史查询、密码维护 |
| 市级用户 | 数据初审、通知发布 | 本市数据审核、通知发布、密码修改 |
| 省级用户 | 数据复核、汇总分析、系统管理 | 全省数据审核、多维分析、系统配置、时限管理 |
| 部委用户 | 数据查看 | 全国汇总数据查看 |

### 2.2 业务流程

#### 2.2.1 企业备案流程
```
省局创建企业账号 → 企业登录 → 补充基础信息 → 提交备案 → 省局审核 → 备案完成
```

#### 2.2.2 月度数据填报流程
```
企业填报数据 → 数据校验 → 提交市局 → 市级初审 → 省级复核 → 汇总上报部委
```

#### 2.2.3 数据分析流程
```
数据采集 → 数据清洗 → 统计分析 → 可视化展示 → 报表导出
```

## 3. 功能需求

### 3.1 企业端功能

#### 3.1.1 企业备案模块
**功能描述:** 企业基础信息录入和备案

**输入项:**
| 字段 | 类型 | 必填 | 校验规则 |
|------|------|------|---------|
| 所属地区 | 下拉选择 | 是 | 显示地市、区域,不可修改 |
| 组织机构代码 | 文本 | 是 | 字母+数字,≤9位 |
| 企业性质 | 两级下拉 | 是 | 从字典选择 |
| 行业分类 | 两级下拉 | 是 | 从字典选择 |
| 邮政编码 | 文本 | 是 | 6位数字 |
| 联系电话 | 文本 | 是 | 符合电话格式 |
| 传真号码 | 文本 | 是 | 符合传真格式 |

**输出:** 备案信息提交成功/失败提示

**业务规则:**
- 组织机构代码必须唯一
- 备案信息提交后需省局审核
- 审核通过后方可进行数据填报

#### 3.1.2 月度填报模块
**功能描述:** 月度就业失业数据填报

**输入项:**
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 建档期就业人数 | 数字 | 是 | 基准人数 |
| 调查期就业人数 | 数字 | 是 | 当前统计人数 |
| 就业人数减少类型 | 下拉选择 | 条件必填 | 调查期<建档期时必填 |
| 主要原因及说明 | 文本 | 条件必填 | 调查期<建档期时必填 |

**校验规则:**
- 人数必须为非负整数
- 如果调查期 < 建档期,必须填写减少类型和原因

**减少类型字典:**
- 产业结构调整
- 重大技术改革
- 关闭破产
- 经济性裁员
- 订单不足
- 用工成本上升
- 自然减员
- 季节性用工
- 自行离职
- 其他

#### 3.1.3 状态查询模块
**功能描述:** 查看历史报送状态

**查询条件:**
- 时间段
- 审核状态(待审核/审核中/已通过/已退回)

**输出:** 报送记录列表,包含填报时间、审核状态、审核意见

#### 3.1.4 密码维护
**功能描述:** 修改登录密码

**要求:**
- 密码长度8-20位
- 包含字母和数字
- 不能与最近3次密码相同

### 3.2 市级端功能

#### 3.2.1 数据审核
**功能描述:** 对企业上报数据进行初审

**操作:**
- 查看企业填报数据
- 审核通过
- 审核退回(需填写退回原因)

**审核标准:**
- 数据格式正确
- 必填项完整
- 数据逻辑合理

#### 3.2.2 通知发布
**功能描述:** 向本市企业发布通知

**通知类型:**
- 填报通知
- 政策通知
- 系统维护通知

### 3.3 省级端功能

#### 3.3.1 企业备案审核
**功能描述:** 审核企业备案信息

**操作:**
- 查看备案信息
- 审核通过
- 审核退回

#### 3.3.2 报表管理
**功能描述:** 管理报表审核、退回、上报

**操作流程:**
- 查看市级审核通过的报表
- 省级复核
- 汇总上报部委

#### 3.3.3 数据修改与汇总
**功能描述:** 数据修正和汇总统计

**权限:** 省级用户可修正错误数据

**汇总维度:**
- 按地区
- 按行业
- 按企业性质

#### 3.3.4 多维分析
**功能描述:** 数据可视化分析

**分析类型:**

1. **对比分析**
   - 折线图对比两个调查期岗位变动
   - 对比指标:企业总数、变动占比

2. **趋势分析**
   - 多个连续调查期岗位变化趋势
   - 占比趋势展示

3. **取样分析**
   - 各市企业数量饼图
   - 企业占比分布

#### 3.3.5 系统管理

**3.3.5.1 上报时限设置**
- 新增调查期时间范围
- 修改调查期时间范围
- 时限提醒设置

**3.3.5.2 用户与权限管理**
- 角色定义
- 功能权限分配
- 用户关联角色

**3.3.5.3 系统监控**
- 系统资源监控
- 用户操作日志
- 异常告警

### 3.3.6 通知发布
**功能描述:** 向全省发布通知

## 4. 接口需求

### 4.1 国家系统对接接口
**接口描述:** 与国家失业监测系统进行数据交换

**数据内容:**
- 汇总后的就业失业数据
- 企业备案信息

**数据格式:** JSON/XML

**交换频率:** 每月一次

### 4.2 数据导出接口
**导出格式:**
- Excel
- PDF
- CSV

## 5. 非功能需求

### 5.1 性能需求
- 页面响应时间: ≤3秒
- 报表生成时间: ≤10秒
- 支持并发用户: ≥500
- 大数据量汇总: ≤30秒

### 5.2 安全需求
- 数据传输加密(HTTPS)
- 密码加密存储
- 操作日志记录
- 会话超时自动退出(30分钟)

### 5.3 可用性需求
- 系统可用性: ≥99.9%
- 支持7×24小时访问
- 计划维护提前通知

### 5.4 兼容性需求
- 浏览器: Chrome 80+, Firefox 75+, Edge 80+
- 分辨率: ≥1366×768
- 移动端适配(可选)

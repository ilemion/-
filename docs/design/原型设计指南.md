# 原型设计指南

## 1. 设计原则

### 1.1 用户体验原则
- **简洁明了:** 减少用户学习成本,表单填写简单直观
- **一致性:** 保持界面风格和操作方式一致
- **容错性:** 提供清晰的错误提示和引导
- **响应性:** 操作反馈及时,避免长时间等待

### 1.2 政府系统特殊要求
- **正式性:** 界面风格正式、专业
- **可访问性:** 支持不同年龄段用户使用
- **可追溯性:** 操作记录完整,可审计

## 3. 核心页面设计

### 3.1 企业备案页面

**页面名称:** 企业备案信息录入

**布局:**
```
┌────────────────────────────────────┐
│ 企业备案信息                         │
├────────────────────────────────────┤
│                                    │
│ 所属地区: [云南省 ▾] [昆明市 ▾]    │
│                                    │
│ 组织机构代码: [____________]       │
│                                    │
│ 企业性质: [国有企业 ▾]             │
│           [国有企业 ▾]             │
│                                    │
│ 行业分类: [制造业 ▾]               │
│           [电子信息制造业 ▾]       │
│                                    │
│ 邮政编码: [______]                 │
│                                    │
│ 联系电话: [____________]           │
│                                    │
│ 传真号码: [____________]           │
│                                    │
├────────────────────────────────────┤
│            [保存] [提交]           │
└────────────────────────────────────┘
```

**交互说明:**
- 所属地区: 系统自动填充,不可修改
- 组织机构代码: 实时校验,显示格式提示
- 企业性质/行业: 两级联动下拉选择
- 必填项: 用红色星号(*)标注
- 保存: 暂存草稿,可后续继续编辑
- 提交: 提交备案审核,不可再修改

## 4. 表单设计指南

### 4.1 输入框设计
- **宽度:** 根据内容长度自适应
- **占位符:** 提供格式示例
- **实时校验:** 输入时即时提示
- **错误样式:** 红色边框+错误提示

### 4.2 下拉选择设计
- **两级联动:** 选择一级后动态加载二级
- **搜索功能:** 选项超过20个时支持搜索
- **默认选项:** 显示"请选择"

### 4.3 按钮设计
- **主操作按钮:** 蓝色,右侧对齐
- **次操作按钮:** 灰色,主按钮左侧
- **危险操作按钮:** 红色,如"删除"
- **加载状态:** 显示loading动画

### 4.4 表格设计
- **表头:** 固定,背景色区分
- **斑马纹:** 奇偶行不同背景色
- **悬停效果:** 鼠标悬停行高亮
- **分页:** 每页20条数据
- **排序:** 支持点击表头排序

## 5. 响应式设计

### 5.1 断点设置
- **桌面端:** ≥1200px
- **平板端:** 768px-1199px
- **移动端:** <768px(可选)

### 5.2 适配策略
- 表格横向滚动
- 表单纵向排列
- 导航折叠为汉堡菜单

## 8. 交互反馈

### 8.1 成功反馈
- 成功提示: 绿色提示框,2秒后自动关闭
- 操作成功: "操作成功"

### 8.2 错误反馈
- 错误提示: 红色提示框,需手动关闭
- 表单错误: 字段下方红色文字提示

### 8.3 加载状态
- 页面加载: 顶部进度条
- 数据加载: 骨架屏或loading动画
- 按钮加载: 按钮内显示loading

## 9. 无障碍设计

### 9.1 键盘操作
- Tab键切换焦点
- Enter键确认操作
- Esc键取消操作

### 9.2 屏幕阅读器
- 为图标添加alt属性
- 为表单元素添加label
- 为操作添加aria属性

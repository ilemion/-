# 云南省企业就业失业数据采集系统工作说明书

## 1. 引言与用户分析

本文档记录了云南省企业数据采集系统的业务需求。

**系统用户包括：**
- 企业用户
- 省用户
- 市用户

## 2. 流程需求

### 账号与备案
- 省管理部门创建企业账号
- 企业登录并补充基础信息上报省局备案

### 数据上报流转
- 企业每月在规定时间内上报数据至市局
- 市局审核后上报省局
- 省局汇总后上报部委

### 分析与展示
- 省局对数据进行汇总、图形化分析并支持下载

## 3. 功能需求概要

### 业务功能列表

**企业端**
- 基本信息修改
- 数据填报
- 历史状态查询
- 密码修改

**省级端**
- 企业备案审核
- 报表管理（审核/退回/上报）
- 数据修改与汇总
- 多维分析
- 图表分析
- 通知发布
- 系统管理（时限设置、用户与权限管理、系统监控）

**市级端**
- 数据审核
- 通知发布
- 密码修改

## 4. 详细需求说明

### 企业备案信息数据项

| 数据项 | 必填 | 说明 |
|--------|------|------|
| 所属地区 | 是 | 显示所属地市、区域，不可修改 |
| 组织机构代码 | 是 | 字母、数字，不超过9位 |
| 企业性质/行业 | 是 | 两级下拉选择 |
| 邮政编码 | 是 | 6位数字 |
| 联系电话/传真 | 是 | 需符合格式规范 |

### 数据填报说明

**核心指标**
- 建档期就业人数
- 调查期就业人数

**必填逻辑**
- 如果调查期数据小于建档期，则"就业人数减少类型"和"主要原因及说明"为必填项

**减少类型示例**
- 关闭破产
- 经济性裁员
- 自然减员
- 等

### 分析与汇总功能

**对比分析**
- 使用折线图、表格对比两个调查期的岗位变动
- 包含：企业总数、变动占比等

**趋势分析**
- 展示多个连续调查期内的岗位变化占比趋势

**取样分析**
- 显示各市企业数量及占比的饼图

## 5. 系统管理与接口

### 权限管理
- 支持角色定义
- 功能权限分配
- 用户关联

### 上报时限
- 省级用户可新增或修改调查期的时间范围

### 数据接口
- 实现与国家失业监测系统的数据交换

## 6. 字典参考

### 就业人数减少原因

- 产业结构调整
- 重大技术改革
- 订单不足
- 用工成本上升
- 季节性用工
- 自行离职

# 贡献指南

感谢你对云南省企业就业失业数据采集系统的关注!我们欢迎任何形式的贡献。

## 如何贡献

### 报告Bug
1. 使用GitHub的Issues功能报告Bug
2. 使用[Bug报告模板](.github/ISSUE_TEMPLATE/bug_report.md)
3. 尽可能提供详细信息,包括:
   - 问题描述
   - 复现步骤
   - 期望行为
   - 截图(如果适用)
   - 环境信息

### 提出新功能
1. 使用GitHub的Issues功能提出新功能
2. 使用[功能请求模板](.github/ISSUE_TEMPLATE/feature_request.md)
3. 描述:
   - 功能的具体需求
   - 使用场景
   - 期望的实现方案

### 提交代码
1. Fork本仓库
2. 创建你的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交你的修改 (`git commit -m 'feat: add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 创建一个Pull Request

## 提交信息规范
我们使用约定式提交(Conventional Commits):
- `feat:` 新功能
- `fix:` 修复Bug
- `docs:` 文档更新
- `style:` 代码格式调整
- `refactor:` 代码重构
- `test:` 测试相关
- `chore:` 构建过程或辅助工具的变动

## 文档贡献
- 完善项目文档
- 更新README
- 编写API文档
- 添加使用示例

## 行为准则
- 尊重他人
- 建设性地讨论问题
- 接受建设性批评
- 关注对社区最有利的事情

## 如何获得帮助
- 查看Issues中是否已有类似问题的讨论
- 创建新的Issue
- 在Pull Request中@维护者

再次感谢你的贡献!
